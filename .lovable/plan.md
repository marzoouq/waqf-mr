

# إضافة تصدير PDF وطباعة لجدول الوحدات + تحسين التوافق مع الجوال

## الوضع الحالي

### مشاكل التوافق مع الشاشات:
- مربع الحوار بعرض `max-w-6xl` يتجاوز شاشة الجوال
- الجدول يحتوي 10 أعمدة بدون تمرير أفقي مخصص للجوال
- أزرار الدفعات وخلايا الجدول تتراكم على الشاشات الصغيرة

### ما ينقص:
- لا توجد أزرار طباعة أو تصدير PDF داخل مربع حوار الوحدات
- لا توجد دالة `generateUnitsPDF` في ملف `pdfGenerator.ts`

---

## ما سيتم تنفيذه

### 1. إضافة دالة `generateUnitsPDF` في `src/utils/pdfGenerator.ts`

دالة جديدة تصدر تقرير وحدات عقار معين بنفس التصميم الرسمي المتبع (إطار مزدوج، رأس بالشعار، تذييل):

- **العنوان:** "تقرير الوحدات السكنية - عقار [الرقم]"
- **الأعمدة:** رقم الوحدة، النوع، الحالة، المستأجر، بداية العقد، نهاية العقد، الإيجار الشهري، الإيجار السنوي، الدفعات
- **صف إجمالي** لمجموع الإيجارات الشهرية والسنوية
- تستقبل بيانات الوحدات والمستأجرين وبيانات الوقف

### 2. إضافة أزرار الطباعة والتصدير في `src/pages/dashboard/PropertiesPage.tsx`

- زر **"تصدير PDF"** وزر **"طباعة"** بجانب زر "إضافة وحدة" في رأس مربع الحوار
- زر الطباعة يستدعي `window.print()`
- زر التصدير يستدعي `generateUnitsPDF` مع البيانات الحالية
- استيراد `usePdfWaqfInfo` لتمرير بيانات هوية الوقف

### 3. تحسين التوافق مع شاشة الجوال

- تغيير عرض الحوار إلى `max-w-6xl w-[95vw]` لضمان عدم تجاوز الشاشة
- إضافة `overflow-x-auto` على حاوية الجدول لتفعيل التمرير الأفقي على الشاشات الصغيرة
- إضافة `min-w-[800px]` على الجدول نفسه لمنع ضغط الأعمدة
- تحويل أزرار الرأس (إضافة + طباعة + تصدير) إلى تخطيط مرن يلتف `flex-wrap` على الجوال

### 4. تحسينات الطباعة عبر CSS

- إضافة قواعد `@media print` لإخفاء أزرار الإجراءات والعناصر غير الضرورية أثناء الطباعة
- ضمان ظهور الجدول كاملا عند الطباعة

## التفاصيل التقنية

### الملفات المعدلة:

1. **`src/utils/pdfGenerator.ts`** - إضافة دالة `generateUnitsPDF` (حوالي 80 سطر) تتبع نفس نمط الدوال الموجودة
2. **`src/pages/dashboard/PropertiesPage.tsx`** - إضافة أزرار التصدير/الطباعة + تحسين التوافق مع الجوال

### واجهة دالة `generateUnitsPDF`:

```text
المدخلات:
- propertyNumber: string (رقم العقار)
- propertyLocation: string (موقع العقار) 
- units: مصفوفة تحتوي بيانات كل وحدة مع المستأجر والدفعات
- waqfInfo: PdfWaqfInfo (بيانات الرأس والتذييل)

المخرجات:
- ملف PDF يتم تحميله تلقائيا
```

### هيكل التوافق مع الجوال:

```text
الحوار
  └── max-w-6xl w-[95vw] (لا يتجاوز الشاشة)
      └── أزرار الرأس: flex-wrap gap-2
      └── حاوية الجدول: overflow-x-auto
          └── الجدول: min-w-[800px] (يتمرر أفقيا على الجوال)
```
