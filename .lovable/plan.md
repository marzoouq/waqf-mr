
# تطوير لوحة المستفيد الرئيسية - تصميم احترافي

## التعديلات المطلوبة

### 1. مربع ترحيب احترافي (Welcome Card)
استبدال قسم الترحيب الحالي (نص بسيط) بمربع بطاقة احترافي يتضمن:
- **اسم المستفيد** بخط كبير واضح
- **تحية ديناميكية**: "صباح الخير" (قبل 12 ظهراً) أو "مساء الخير" (بعد 12 ظهراً)
- **التاريخ الهجري والميلادي** مع اسم اليوم (مثل: الخميس 20 شعبان 1447هـ - 20 فبراير 2026م)
- **الوقت الحالي** يتحدث تلقائياً كل دقيقة
- أيقونة شمس/قمر حسب الفترة (صباح/مساء)
- تدرج لوني أنيق للخلفية

### 2. تحسينات التجاوب (Responsive)
- على الجوال: المربع يأخذ العرض الكامل مع حجم خط أصغر
- على الشاشات المتوسطة والكبيرة: تخطيط أفقي مع توزيع المعلومات

### 3. مميزات احترافية إضافية
- **شريط تقدم دائري** لنسبة الحصة (بدلاً من رقم فقط)
- **بطاقة حالة السنة المالية** مع عداد الأيام المتبقية
- **قسم آخر التوزيعات** يعرض آخر 3 توزيعات تمت

## التفاصيل التقنية

### الملفات المتأثرة:
- `src/pages/beneficiary/BeneficiaryDashboard.tsx` - تعديل شامل

### منطق التحية:
```typescript
const now = new Date();
const hour = now.getHours();
const greeting = hour < 12 ? 'صباح الخير' : 'مساء الخير';
const greetingIcon = hour < 12 ? Sun : Moon;
```

### منطق التاريخ والوقت:
```typescript
// التاريخ الهجري
const hijriDate = now.toLocaleDateString('ar-SA-u-ca-islamic', {
  weekday: 'long', year: 'numeric', month: 'long', day: 'numeric'
});
// التاريخ الميلادي
const gregorianDate = now.toLocaleDateString('ar-SA', {
  year: 'numeric', month: 'long', day: 'numeric'
});
// الوقت (يتحدث كل دقيقة عبر setInterval)
const time = now.toLocaleTimeString('ar-SA', {
  hour: '2-digit', minute: '2-digit'
});
```

### هيكل مربع الترحيب الجديد:

|  الخميس 20 شعبان 1447هـ  |  20 فبراير 2026م  | 10:30 |
|  واجهة المستفيد                                       |
+------------------------------------------------------+

### بطاقة شريط تقدم الحصة:
دائرة SVG تعرض نسبة حصة المستفيد بصرياً مع الرقم في المنتصف، بدلاً من عرض النسبة كرقم فقط.

### بطاقة السنة المالية:
تعرض اسم السنة المالية الحالية مع شريط تقدم يوضح كم مضى من السنة وكم تبقى.
