

# تحسين صفحة العقود - ربط بيانات الوحدات والدفعات

## ما تم إنجازه سابقاً
- حقول الإيجار والدفعات في نموذج إضافة العقد (تعمل)
- حقول الإيجار في نموذج الوحدات عند اختيار "مؤجرة" (تعمل)
- خيار "العقار كامل" في صفحة العقارات (يعمل)

## ما لم يُنفذ بعد

### 1. جدول العقود لا يعرض معلومات الدفع والوحدة
الجدول الحالي يعرض 8 أعمدة فقط ولا يتضمن:
- عمود **الوحدة** (أي وحدة مرتبطة بالعقد أو "العقار كامل")
- عمود **نوع الدفع** (سنوي / شهري / متعدد)
- عمود **قيمة الدفعة** (المبلغ لكل دفعة)

### 2. نموذج العقد لا يحتوي على حقل اختيار الوحدة
عند إضافة عقد من صفحة العقود، لا يوجد خيار لربطه بوحدة محددة.

### 3. استعلام العقود لا يجلب بيانات الوحدة
`useContracts` يجلب فقط `property:properties(*)` بدون `unit:units(*)`.

## الإصلاحات

### تعديل 1: تحديث استعلام العقود (`src/hooks/useContracts.ts`)
تعديل الاستعلام ليشمل بيانات الوحدة:
```text
قبل: .select('*, property:properties(*)')
بعد: .select('*, property:properties(*), unit:units(*)')
```

### تعديل 2: إضافة أعمدة جديدة في جدول العقود (`src/pages/dashboard/ContractsPage.tsx`)
إضافة 3 أعمدة بعد عمود "العقار":
- **الوحدة**: تعرض رقم الوحدة أو "العقار كامل" إذا لم يكن هناك unit_id
- **نوع الدفع**: سنوي / شهري / متعدد (X دفعات)
- **قيمة الدفعة**: المبلغ المحسوب

### تعديل 3: إضافة حقل اختيار الوحدة في نموذج العقد
بعد حقل اختيار العقار:
- جلب وحدات العقار المختار عبر `useUnits(property_id)`
- عرض قائمة منسدلة تحتوي على:
  - "العقار كامل" كخيار أول
  - قائمة وحدات العقار المحدد
- تعيين `unit_id` عند اختيار وحدة

### تعديل 4: تحديث نوع Contract (`src/types/database.ts`)
إضافة حقل `unit` الاختياري لواجهة Contract لدعم بيانات الوحدة المجلوبة.

### تعديل 5: زيادة min-width الجدول
تحديث `min-w-[750px]` إلى `min-w-[1000px]` لاستيعاب الأعمدة الجديدة.
