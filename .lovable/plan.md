## التفاصيل التقنية

### تحديث hooks/useContracts.ts
```text
إضافة useUpdateContract mutation:
- يستخدم supabase.update()
- يُبطل cache بعد النجاح
- يظهر toast للنجاح/الفشل
```

### تحديث hooks/useIncome.ts
```text
إضافة useUpdateIncome mutation:
- نفس النمط السابق
```

### تحديث hooks/useExpenses.ts
```text
إضافة useUpdateExpense mutation:
- نفس النمط السابق
```

### صفحة الإفصاح السنوي للمستفيد
```text
- عرض تفصيلي للإفصاح السنوي
- جدول بالإيرادات والمصروفات
- حساب الحصة تلقائياً
- زر تحميل PDF للإفصاح
```

### صفحة حصتي من الريع
```text
- عرض الحصة الحالية
- سجل التوزيعات السابقة
- حالة الدفع (مستلم/معلق)
```

### صفحة التقارير المالية للمستفيد
```text
- رسوم بيانية مبسطة
- مقارنة سنوية
- تصدير PDF
```
