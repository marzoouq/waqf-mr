

# تنظيم صفحة الفواتير

## المشاكل المكتشفة
1. بطاقات الملخص (InvoiceSummaryCards) موضوعة **فوق العنوان** (سطر 179) بينما في كل الصفحات الأخرى العنوان يأتي أولاً
2. شريط التبديل (جدول/شبكي) في صف منفصل مع `<div></div>` فارغ بجانبه (سطر 287)
3. حقل البحث في صف ثالث منفصل (سطر 294)
4. لا توجد فلاتر للنوع أو الحالة
5. لا يوجد تحذير السنة المقفلة
6. لا يوجد عرض بطاقات مخصص للجوال في وضع الجدول

## التغييرات المطلوبة (ملف واحد فقط)

**`src/pages/dashboard/InvoicesPage.tsx`**

### 1. إعادة ترتيب التخطيط
- نقل العنوان + الأزرار إلى الأعلى
- إضافة تحذير السنة المقفلة (مثل الصفحات الأخرى)
- نقل بطاقات الملخص بعد العنوان

### 2. دمج شريط الأدوات في صف واحد
- حقل البحث + فلتر النوع (إيجار/صيانة/خدمات/أخرى) + فلتر الحالة (مدفوعة/معلقة/ملغاة) + تبديل العرض -- كلها في صف واحد متجاوب

### 3. إضافة فلاتر النوع والحالة
- إضافة state جديدة: `filterType` و `filterStatus`
- تحديث `filteredInvoices` لتطبيق الفلاتر الثلاثة (بحث + نوع + حالة)

### 4. إضافة عرض بطاقات للجوال
- في وضع الجدول، إضافة عرض بطاقات `md:hidden` يظهر على الشاشات الصغيرة
- كل بطاقة تعرض: النوع، الرقم، المبلغ، التاريخ، الحالة (Badge)، أزرار (عرض/تعديل/حذف)

### الترتيب النهائي للصفحة
```text
1. العنوان + أزرار الإجراءات
2. تحذير السنة المقفلة (إن وُجد)
3. بطاقات الملخص
4. شريط الأدوات: بحث | فلتر النوع | فلتر الحالة | تبديل العرض
5. الجدول (desktop) / بطاقات (mobile) أو العرض الشبكي
6. التصفح (Pagination)
```

لا توجد تغييرات في قاعدة البيانات أو ملفات أخرى.
