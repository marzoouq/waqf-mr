

## إضافة شريط تقدم مرئي لحالة التحصيل

### الفكرة
تحسين عرض التحصيل في صفحة العقود بإضافة شريط تقدم (Progress bar) ملون أسفل أرقام الدفعات، مشابه لما هو موجود حاليا في صفحة العقارات. الشريط يتغير لونه حسب نسبة التحصيل.

### التعديلات

**تعديل ملف واحد:** `src/pages/dashboard/ContractsPage.tsx`

1. استيراد مكون `Progress` من `@/components/ui/progress`
2. في عمود التحصيل بالجدول (سطر 497-508): إضافة شريط Progress أسفل أزرار +/- مع الأرقام
3. في بطاقات الموبايل (سطر 450-470): إضافة شريط Progress أسفل أزرار التحصيل
4. تلوين الشريط:
   - **أخضر**: إذا اكتملت جميع الدفعات (paid >= paymentCount)
   - **أصفر**: إذا تم دفع نصف الدفعات أو أكثر
   - **أحمر**: إذا تم دفع أقل من النصف

### القسم التقني

**الملف:** `src/pages/dashboard/ContractsPage.tsx`

- إضافة `import { Progress } from '@/components/ui/progress'`
- في الجدول: إضافة `<Progress>` أسفل div الأزرار بنسبة `(paid / paymentCount) * 100`
- في بطاقات الموبايل: نفس الإضافة
- التلوين عبر classes ديناميكية: `[&>div]:bg-success` / `[&>div]:bg-warning` / `[&>div]:bg-destructive`
