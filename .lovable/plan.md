

# إضافة خيار تأجير العقار كامل وحقول الإيجار والدفعات

## المشكلة الحالية

1. **لا يوجد خيار لتأجير العقار كامل**: النظام يفترض أن كل عقار مقسم لوحدات، لكن بعض العقارات تُؤجر كوحدة واحدة (مثل فيلا أو أرض).
2. **نموذج الوحدات لا يحتوي على حقول الإيجار**: عند إضافة وحدة لا يمكن تحديد قيمة الإيجار أو نوع الدفع أو عدد الدفعات - هذه البيانات تأتي فقط من جدول العقود المنفصل.
3. **الطباعة**: يجب التحقق من أن إصلاحات CSS السابقة تعمل بشكل صحيح.

---

## ما سيتم تنفيذه

### 1. إضافة خيار "تأجير العقار كامل" في صفحة العقارات

عند الضغط على عقار، يظهر خياران في أعلى مربع الحوار:
- **وحدات منفصلة** (الوضع الحالي): إدارة الوحدات كل واحدة على حدة
- **العقار كامل**: عرض عقد واحد مرتبط بالعقار مباشرة (بدون وحدات)

عند اختيار "العقار كامل":
- يختفي جدول الوحدات ويظهر بدلاً منه معلومات العقد المرتبط بالعقار (المستأجر، الإيجار، الدفعات)
- إذا لم يكن هناك عقد، يظهر زر "إضافة عقد للعقار" يوجه لصفحة العقود

### 2. إضافة حقول الإيجار في نموذج الوحدات

عند إضافة أو تعديل وحدة بحالة "مؤجرة"، تظهر حقول إضافية:
- **اسم المستأجر**: حقل نصي
- **قيمة الإيجار السنوي**: حقل رقمي
- **نوع الدفع**: قائمة منسدلة (سنوي / شهري / دفعات متعددة)
- **عدد الدفعات**: يظهر عند اختيار "دفعات متعددة"
- **قيمة الدفعة**: تحسب تلقائياً (الإيجار السنوي / عدد الدفعات)

عند الحفظ:
- يتم إنشاء عقد جديد تلقائياً في جدول `contracts` مرتبط بالوحدة
- أو تحديث العقد الموجود إذا كانت الوحدة مؤجرة مسبقاً

### 3. التحقق من الطباعة

- مراجعة قواعد CSS للطباعة في `index.css`
- التأكد من أن محتوى مربع الحوار يظهر عند الطباعة

## التفاصيل التقنية

### الملفات المعدلة:

1. **`src/pages/dashboard/PropertiesPage.tsx`**
   - إضافة تبويب "العقار كامل / وحدات منفصلة" في `PropertyUnitsDialog`
   - إضافة حقول الإيجار المشروطة (تظهر عند حالة "مؤجرة") في نموذج الوحدات
   - منطق إنشاء/تحديث العقد تلقائياً عند حفظ وحدة مؤجرة

2. **`src/hooks/useContracts.ts`** - التحقق من وجود دالة إنشاء عقد يمكن استخدامها

### منطق نموذج الوحدة المحسّن:

```text
عند اختيار الحالة = "مؤجرة":
  -> تظهر حقول: اسم المستأجر، الإيجار، نوع الدفع، عدد الدفعات
  -> عند الحفظ:
     1. حفظ الوحدة في جدول units
     2. إنشاء عقد في جدول contracts مرتبط بـ unit_id و property_id
     3. تعيين payment_type و payment_count و payment_amount

عند اختيار الحالة = "شاغرة" أو "صيانة":
  -> تختفي حقول الإيجار
  -> تُحفظ الوحدة فقط بدون عقد
```

### عرض "العقار كامل":

```text
البحث في العقود: contracts.filter(c => c.property_id === property.id && !c.unit_id)
  -> إذا وُجد عقد: عرض بيانات المستأجر والإيجار والدفعات
  -> إذا لم يوجد: عرض رسالة "لا يوجد عقد" + زر إضافة
```
