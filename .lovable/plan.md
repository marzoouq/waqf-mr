

# إعادة تنظيم صفحة العقود — تجميع العقود المتجددة والدفعات

## المشكلة الحالية

صفحة العقود تعرض كل عقد كصف منفصل — عقد أصلي `C-001` وتجديداته `C-001-R1`, `C-001-R2` كلها صفوف مستقلة. مع تراكم السنوات يصبح الجدول طويلاً جداً ويصعب تتبع تاريخ العقد الواحد. كذلك الدفعات (فواتير الدفعات) في تبويب منفصل بدون ربط مباشر بالعقد.

## الحل المقترح

### 1. تجميع العقود حسب رقم العقد الأساسي (Accordion)

```text
┌─────────────────────────────────────────────────────┐
│ ▼ عقد C-001 | مستأجر: أحمد | عقار: 101 | نشط      │
│   ┌─────────────────────────────────────────────────┤
│   │ C-001    | 2023-01-01 → 2023-12-31 | منتهي     │
│   │ C-001-R1 | 2024-01-01 → 2024-12-31 | منتهي     │
│   │ C-001-R2 | 2025-01-01 → 2025-12-31 | نشط  ✏️🗑 │
│   ├─────────────────────────────────────────────────┤
│   │ 📄 الدفعات (2025):                              │
│   │   دفعة 1: 15,000 ر.س ✅ مسددة                  │
│   │   دفعة 2: 15,000 ر.س ⏳ معلقة                  │
│   │   دفعة 3: 15,000 ر.س ❌ متأخرة                 │
│   └─────────────────────────────────────────────────┤
│                                                     │
│ ▶ عقد C-002 | مستأجر: محمد | عقار: 102 | نشط      │
│                                                     │
│ ▶ عقد C-003 | مستأجر: خالد | عقار: 103 | منتهي    │
└─────────────────────────────────────────────────────┘
```

- العقود تُجمَّع تلقائياً بإزالة `-R1`, `-R2` من رقم العقد للوصول للرقم الأساسي
- الصف الرئيسي يعرض: الرقم الأساسي، اسم المستأجر، العقار، حالة آخر عقد، إجمالي الإيجار الحالي
- بالضغط ينسدل: جدول مصغّر بكل إصدارات العقد + قسم الدفعات للعقد النشط

### 2. الدفعات مدمجة تحت كل عقد

بدلاً من تبويب منفصل للفواتير، تظهر الدفعات مباشرة تحت العقد النشط في القائمة المنسدلة. تبويب "فواتير الدفعات" يبقى متاحاً كعرض شامل لكن العرض الأساسي يكون مدمجاً.

### 3. المنطق التقني

- **التجميع**: `useMemo` يبني `Map<baseNumber, Contract[]>` بتقسيم `contract_number` على `-R`
- **الترتيب**: المجموعات تُرتَّب حسب أحدث `start_date`، والعقود داخل المجموعة حسب التاريخ تنازلياً
- **الدفعات**: `paymentInvoices` مُفلترة بـ `contract_id` للعقد النشط تظهر كجدول مصغّر
- **مكوّن جديد**: `ContractAccordionGroup.tsx` — يستقبل مجموعة عقود + فواتير ويعرضها كـ Collapsible
- **البحث/الفلترة**: تعمل على مستوى المجموعة (تبحث في كل العقود ضمنها)
- **التصدير والإحصائيات**: تبقى كما هي بدون تغيير
