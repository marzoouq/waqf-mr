

# تحويل شريط Beta إلى مكون ديناميكي يتحكم فيه الناظر

## المشكلة الحالية
- الشريط موضوع في `App.tsx` خارج تخطيط الصفحات مما يجعله يغطي الهيدر العلوي (تناقض بصري)
- النص ثابت في الكود ولا يمكن للناظر تعديله

## الحل المقترح

### 1. تخزين إعدادات الشريط في قاعدة البيانات
إضافة مفتاح جديد `beta_banner_settings` في جدول `app_settings` بالقيم التالية:

```text
{
  "enabled": true,              // إظهار/إخفاء الشريط
  "text": "إصدار تجريبي — نعمل على تطويره من أجلكم ونرحب بملاحظاتكم",
  "color": "amber",             // اللون: amber, blue, green, red, purple
  "position": "top",            // الموقع: top (أعلى المحتوى) أو bottom (أسفل)
  "dismissible": true           // هل يمكن للمستخدم إخفاؤه مؤقتاً
}
```

### 2. نقل الشريط من App.tsx إلى DashboardLayout.tsx
- ازالته من `App.tsx` (حيث يغطي الهيدر)
- وضعه **داخل** `DashboardLayout` فوق المحتوى الرئيسي مباشرة (بعد الهيدر)
- بهذا يظهر الشريط في جميع صفحات لوحة التحكم دون أن يغطي الهيدر

### 3. تحديث مكون BetaBanner.tsx
- يقرأ الإعدادات من `app_settings` عبر `useAppSettings`
- يدعم 5 ألوان: ذهبي، أزرق، أخضر، أحمر، بنفسجي
- يدعم موقعين: أعلى المحتوى أو أسفل الصفحة
- زر إغلاق مؤقت (اختياري حسب إعداد الناظر)

### 4. إضافة تبويب "شريط التنبيه" في صفحة الإعدادات
تبويب جديد في `SettingsPage.tsx` يتيح للناظر:
- تفعيل/تعطيل الشريط بمفتاح Switch
- تغيير نص الشريط
- اختيار اللون من 5 خيارات
- اختيار الموقع (أعلى/أسفل)
- تفعيل/تعطيل زر الإغلاق

---

## الملفات المتأثرة

| الملف | التغيير |
|-------|---------|
| `src/components/BetaBanner.tsx` | إعادة كتابة ليقرأ من الإعدادات ويدعم الألوان والمواقع |
| `src/App.tsx` | إزالة `BetaBanner` |
| `src/components/DashboardLayout.tsx` | إضافة `BetaBanner` داخل التخطيط |
| `src/pages/dashboard/SettingsPage.tsx` | إضافة تبويب "شريط التنبيه" |
| قاعدة البيانات | إضافة صف `beta_banner_settings` في `app_settings` |

## التفاصيل التقنية

### هيكل الألوان المدعومة:
- **ذهبي (amber)**: `bg-amber-500` — للتنبيهات العامة
- **أزرق (blue)**: `bg-blue-500` — للمعلومات
- **أخضر (green)**: `bg-green-600` — للأخبار الإيجابية
- **أحمر (red)**: `bg-red-500` — للتحذيرات المهمة
- **بنفسجي (purple)**: `bg-purple-500` — للإعلانات

### ترتيب العرض في DashboardLayout:
```text
+---------------------------+
| الهيدر العلوي (Header)    |  <-- يبقى كما هو
+---------------------------+
| شريط التنبيه (اختياري)    |  <-- هنا يظهر الشريط
+---------------------------+
| المحتوى الرئيسي           |
+---------------------------+
```
