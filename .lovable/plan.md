

# إصلاح خطأ البناء — عدم توافق Tailwind CSS v4 مع إعدادات v3

## المشكلة

في `package.json` سطر 106:
```
"tailwindcss": "^4.2.2"
```

بينما المشروع يستخدم `tailwind.config.ts` بصيغة **Tailwind v3** (مع `content`, `darkMode`, `prefix`, `theme`). 

Tailwind v4 لا يدعم هذه الصيغة — يستخدم نظام إعداد مختلف تمامًا عبر CSS مباشرة.

هذا يسبب فشل PostCSS أثناء البناء → خطأ 500 → التطبيق لا يعمل.

## الإصلاح

**ملف واحد:** `package.json` — تغيير سطر 106 من:
```json
"tailwindcss": "^4.2.2",
```
إلى:
```json
"tailwindcss": "^3.4.17",
```

هذا يُرجع Tailwind إلى الإصدار المتوافق مع الإعدادات الموجودة.

## الملفات المتأثرة

| ملف | التعديل |
|---|---|
| `package.json` سطر 106 | تثبيت tailwindcss على v3.4.17 |

لا تعديل على أي ملف آخر.
