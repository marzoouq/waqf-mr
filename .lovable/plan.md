

# تشغيل جميع اختبارات المشروع

## الوضع الحالي
المشروع يحتوي على **~80+ ملف اختبار** موزعة على:
- `src/test/` — اختبارات أمنية ومالية وتكامل
- `src/pages/` — اختبارات صفحات (عامة + لوحة تحكم + مستفيد)
- `src/components/` — اختبارات مكونات (Sidebar, accounts, invoices, etc.)
- `src/hooks/` — اختبارات hooks
- `src/utils/` — اختبارات أدوات مساعدة
- `src/constants/` — اختبارات ثوابت

## الخطة
عند التحويل لوضع التنفيذ، سأقوم بـ:

1. **تشغيل جميع الاختبارات** عبر أداة `run_tests` بدون تحديد ملف معين (لتشغيل كامل المجموعة)
2. **فحص النتائج** والتحقق من:
   - عدد الاختبارات الناجحة / الفاشلة / المتخطاة
   - أي تعارضات بين الاختبارات الجديدة والقديمة
3. **إصلاح أي فشل** إن وُجد

لا توجد تعديلات مطلوبة على الكود — فقط تشغيل وفحص.
