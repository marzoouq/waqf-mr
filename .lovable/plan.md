

# إضافة نظام الوحدات السكنية للعقارات

## الوضع الحالي (مؤكد بالدليل)
- عقار "101-102" بالطائف يحتوي على 17 عقد إيجار لمستأجرين مختلفين، لكن لا توجد طبقة "وحدات" تربط كل عقد بشقة محددة
- العقارات الأخرى (جدة 001، 002، 003) ليس لها عقود بعد
- صفحة العقارات تعرض بطاقات ثابتة فقط بدون أي تفاعل

## خطة التنفيذ

### الخطوة 1: Migration لقاعدة البيانات

**إنشاء جدول `units`:**

| العمود | النوع | ملاحظات |
|--------|-------|---------|
| id | uuid | PK |
| property_id | uuid FK | مرتبط بالعقار |
| unit_number | text | رقم الوحدة (شقة 1، محل 3) |
| unit_type | text | شقة / محل / مكتب |
| floor | text | أرضي، أول، ثاني |
| area | numeric | المساحة (اختياري) |
| status | text | شاغرة / مؤجرة / صيانة (افتراضي: شاغرة) |
| notes | text | ملاحظات (اختياري) |

**إضافة عمود `unit_id` لجدول العقود:**
- nullable لأن العقود الحالية (17 عقد) ليس لها وحدات بعد
- FK مع ON DELETE SET NULL (عند حذف الوحدة، يبقى العقد بدون وحدة بدل خطأ)

**سياسات RLS:**
- Admin: إدارة كاملة (ALL)
- أصحاب الأدوار (admin + waqif + beneficiary): قراءة فقط (SELECT)

### الخطوة 2: إنشاء هوك `src/hooks/useUnits.ts`
- `useUnits(propertyId)` - جلب وحدات عقار معين
- `useCreateUnit` - إضافة وحدة
- `useUpdateUnit` - تعديل وحدة
- `useDeleteUnit` - حذف وحدة (مع التحقق من عدم وجود عقود مرتبطة)

### الخطوة 3: إضافة نوع Unit في `src/types/database.ts`

```text
Unit: id, property_id, unit_number, unit_type, floor, area, status, notes, created_at, updated_at
Contract: إضافة unit_id? و unit?: Unit
```

### الخطوة 4: تعديل صفحة العقارات `PropertiesPage.tsx`

عند الضغط على بطاقة العقار، يفتح Dialog كبير يحتوي:
- معلومات العقار في الأعلى
- جدول بالوحدات السكنية: رقم الوحدة، النوع، الدور، الحالة، المستأجر الحالي (من العقود)
- أزرار: إضافة وحدة، تعديل، حذف
- عدّاد يظهر (مؤجرة X / شاغرة Y / إجمالي Z)

### الملفات المتأثرة

| الملف | التغيير |
|-------|---------|
| Migration SQL جديد | إنشاء جدول units + إضافة unit_id للعقود + RLS |
| `src/hooks/useUnits.ts` | ملف جديد - هوك CRUD للوحدات |
| `src/types/database.ts` | إضافة واجهة Unit + تحديث Contract |
| `src/pages/dashboard/PropertiesPage.tsx` | إضافة Dialog تفاصيل العقار مع إدارة الوحدات |

### الاحتياطات المتخذة
- العقود الحالية (17 عقد) لن تتأثر لأن `unit_id` اختياري
- حذف وحدة لن يكسر العقد (ON DELETE SET NULL)
- صفحة العقود الحالية تستمر بالعمل بدون تعديل
