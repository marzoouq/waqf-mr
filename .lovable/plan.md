

# إضافة شرح مرئي لخطوات إصدار الفاتورة الضريبية

## الفكرة
إنشاء مكون `InvoiceStepsGuide` قابل للطي (Collapsible) يعرض خطوات إصدار الفاتورة الضريبية كـ stepper مرئي بأيقونات وألوان واضحة. يُعرض في أعلى تبويب فواتير الدفعات (`PaymentInvoicesTab`) وصفحة ZATCA.

## التصميم المرئي
```text
┌─────────────────────────────────────────────────────────┐
│ 📋 كيف تصدر فاتورة ضريبية؟              [طي/توسيع ▾]  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ①──────②──────③──────④──────⑤──────⑥                  │
│  إنشاء   توليد   توقيع   إبلاغ   اعتماد   تحميل       │
│  العقد   الفواتير رقمي   ZATCA   الهيئة   PDF          │
│                                                         │
│  كل خطوة تحتها وصف مختصر بسطر واحد                     │
└─────────────────────────────────────────────────────────┘
```

## التغييرات

### ملف جديد: `src/components/invoices/InvoiceStepsGuide.tsx`
- مكون Collapsible يعرض 6 خطوات بأيقونات Lucide
- كل خطوة: أيقونة + عنوان + وصف مختصر
- stepper أفقي على desktop، عمودي على mobile
- حالة الطي محفوظة في `localStorage`

### تعديل: `src/components/contracts/PaymentInvoicesTab.tsx`
- إضافة `<InvoiceStepsGuide />` قبل بطاقات الملخص

### تعديل: `src/pages/dashboard/ZatcaManagementPage.tsx`
- إضافة `<InvoiceStepsGuide />` بعد `PageHeaderCard`
