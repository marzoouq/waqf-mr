
# اضافة الطباعة وتصدير PDF وصلاحيات التعديل والحذف لجميع الصفحات

## ملخص التغييرات

اضافة زر **طباعة** وزر **تصدير PDF** في اعلى كل صفحة من صفحات لوحة التحكم، مع التاكد من وجود صلاحيات التعديل والحذف في جميع الاقسام.

## تحليل الوضع الحالي

| الصفحة | اضافة | تعديل | حذف | طباعة | تصدير PDF |
|--------|-------|-------|-----|-------|-----------|
| العقارات | موجود | موجود | موجود | غير موجود | غير موجود |
| العقود | موجود | موجود | موجود | غير موجود | غير موجود |
| الدخل | موجود | موجود | موجود | غير موجود | غير موجود |
| المصروفات | موجود | موجود | موجود | غير موجود | غير موجود |
| المستفيدين | موجود | موجود | موجود | غير موجود | غير موجود |
| الحسابات الختامية | موجود | جزئي | غير موجود | غير موجود | غير موجود |
| التقارير | - | - | - | موجود | موجود |

## التغييرات المطلوبة

### 1. اضافة الطباعة وتصدير PDF لجميع الصفحات (6 صفحات)

لكل صفحة من الصفحات التالية سيتم اضافة:
- زر **طباعة** (يستخدم `window.print()`)
- زر **تصدير PDF** (يستخدم jsPDF + jspdf-autotable مع خط Amiri العربي)

الصفحات: العقارات، العقود، الدخل، المصروفات، المستفيدين، الحسابات الختامية

### 2. اضافة حذف وتعديل للحسابات الختامية

صفحة الحسابات الختامية تحتاج:
- زر حذف للسجلات السابقة (جدول السجلات السابقة)
- اضافة `useDeleteAccount` في `useAccounts.ts`

## التفاصيل التقنية

### الملفات المعدلة:

1. **`src/utils/pdfGenerator.ts`** - اضافة دوال تصدير جديدة:
   - `generatePropertiesPDF(properties)` - جدول العقارات
   - `generateContractsPDF(contracts)` - جدول العقود
   - `generateIncomePDF(income, total)` - جدول الدخل
   - `generateExpensesPDF(expenses, total)` - جدول المصروفات
   - `generateBeneficiariesPDF(beneficiaries)` - جدول المستفيدين
   - `generateAccountsPDF(data)` - الحسابات الختامية الكاملة (عقود + تحصيل + ايرادات + مصروفات + توزيع + مستفيدين)

2. **`src/hooks/useAccounts.ts`** - اضافة `useDeleteAccount`

3. **`src/pages/dashboard/PropertiesPage.tsx`** - اضافة ازرار طباعة + PDF
4. **`src/pages/dashboard/ContractsPage.tsx`** - اضافة ازرار طباعة + PDF
5. **`src/pages/dashboard/IncomePage.tsx`** - اضافة ازرار طباعة + PDF
6. **`src/pages/dashboard/ExpensesPage.tsx`** - اضافة ازرار طباعة + PDF
7. **`src/pages/dashboard/BeneficiariesPage.tsx`** - اضافة ازرار طباعة + PDF
8. **`src/pages/dashboard/AccountsPage.tsx`** - اضافة ازرار طباعة + PDF + حذف السجلات السابقة

### نمط الازرار في كل صفحة:
سيتم اضافة الازرار بجانب زر "اضافة" الموجود في الـ Header:

```text
[طباعة]  [تصدير PDF]  [اضافة عقار]
```

### دالة PDF النموذجية:
كل دالة تصدير ستتبع نفس النمط الموجود في `generateAnnualReportPDF`:
- تحميل خط Amiri العربي
- عنوان الصفحة
- جدول البيانات باستخدام `autoTable`
- تذييل الصفحة بالتاريخ ورقم الصفحة

### حذف السجلات السابقة في الحسابات:
- اضافة `useDeleteAccount` مع mutation يحذف من جدول `accounts`
- اضافة عمود "اجراءات" في جدول السجلات السابقة مع زر حذف
