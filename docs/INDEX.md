## التسلسل المالي (مرجع سريع)

```
إيرادات + رصيد مرحل = إجمالي شامل
  − مصروفات = صافي بعد المصاريف
  − ضريبة = صافي بعد الضريبة
  − زكاة = صافي بعد الزكاة
  − حصة ناظر − حصة واقف = ريع الوقف
  − رقبة وقف = المبلغ المتاح
  − توزيعات = الرصيد المتبقي
```
