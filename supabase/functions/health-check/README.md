# health-check

دالة فحص الحالة (Health Check) للخدمات الخلفية.

## الغرض
نقطة نهاية خفيفة تُرجِع `{ status: "ok" }` للتأكد من أن الـEdge Functions تعمل وأن قاعدة البيانات قابلة للوصول.

## الاستخدام المتوقع
- **مراقبة خارجية** (UptimeRobot / StatusCake / Better Uptime) — استدعاء HTTP GET دوري
- **CI/CD smoke tests** — التحقق بعد النشر من سلامة البيئة
- **تشخيص يدوي** — عند الاشتباه بأعطال في الشبكة أو قاعدة البيانات

## التفعيل
لم يُربط حالياً بأي مراقب خارجي. لربطه:
1. احصل على رابط الدالة: `https://<project>.supabase.co/functions/v1/health-check`
2. أضف الرابط في خدمة المراقبة بفترة 5 دقائق

## الأمان
`verify_jwt = false` — مقصود ليكون متاحاً للمراقبة العامة دون مصادقة.
لا يكشف أي بيانات حساسة.
