# مكونات الوقف (waqf)

مكونات واجهة المستخدم الخاصة بإدارة **الوقف** ككيان — مثل بيانات الوقف، الشعار، والإعدادات العامة.

> **ملاحظة (#31 من الفحص العميق):** `waqf` = الوقف (الكيان المؤسسي)، بينما `waqif` = الواقف (الشخص المانح).
> المجلدان متوازيان عمداً لأنهما يخدمان مفاهيم مختلفة:
> - `waqf/` = إدارة بيانات المؤسسة (Logo, Settings, Bylaws header, ...)
> - `waqif/` = واجهة الواقف كـ **مستخدم** له صلاحيات قراءة على التقارير

لا تخلط بين المجلدين — كل مكوّن يجب أن يبقى في موقعه الدلالي.
